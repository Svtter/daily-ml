LMDB means lightning memory-mapped databases (LMDBs)。 An LMDB dataset can be used to train Caffe models.

使用 LMDB 可以较快的提升训练速度。我之前也构建过类似的 LMDB dataset。

不过这种数据集没有具体的格式要求。也就是说，你可以按照你想要的方式处理 data shape.

lmdb 一般用 lmdb python 包来做管理，通过 pip install lmdb 进行安装。
